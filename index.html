Geometry Digital Portfolio

Made by Anthony Cavallo

The first project that I will be presenting are the SNAP programs introduced in Unit 4. These individual projects are composed of code that, while may seem small and sparse, illustrate the coordinates required to make triangles strictly using code. You can find out the incenter, centroid, orthocenter and the circumcenter of a triangle just by punching in numbers in the block coding. I enjoyed this project because it allowed me to find out these values without having to find them out myself.

The next project that will be shown are the drum beats made in Unit 5. In Groove Pizza, I used shapes and angles to make beats on a circular plate called a groove pizza. After I finish making the groove pizza with the drum beats set, we save it as an audio file that gets automatically imported to Soundtrap so anyone with the link will be able to view and ultimately play the beat. This was a good project because I learned about how angles and shapes make beats and used them to figure out what are the best beats that you can make with just shapes.

The final project that will be presented is the Holes project, which is a project that allows us to find out the packing density of a space with trigonometry formulas and advanced circular formulas. The premise of this project was to figure out how to find the buried treasure in Holes as efficiently as possible and with math we were able to figure out the best way to find the buried treasure. Since we had the knowledge, we were able to complete this project in only a few days. I wonder how long it took to find the buried treasure in Holes. I mean, for them it took them 2 hours (the movie’s length) so I guess we aren’t as fast but if they knew how to use math for their quest then the movie would be quite short to say the least.

And so we have reached the end of my Geometry Digital Portfolio™!
